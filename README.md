# API-BackEnd-CoffeShop

This is API for my backend coffeshop using nodejs and express.

## **to start project**
```
npm start
```

## This project using:
* "body-parser": "^1.19.0",
* "dotenv": "^8.2.0",
* "express": "^4.17.1",
* "morgan": "^1.10.0",
* "pg": "^8.5.1"

## This project have 3 table:
1. Product
2. History
3. Category
